# Bouza Glatigny - Higiene y Seguridad

Sitio web corporativo y plataforma de capacitación para Bouza Glatigny.

## Stack

- **Next.js 15** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS 4**

## Desarrollo

```bash
npm install
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000).

## Estructura

- `src/app/` — Páginas y rutas
- `src/components/` — Componentes reutilizables
- `src/lib/data.ts` — Datos del sitio (contacto, servicios, etc.)
- `public/` — Assets estáticos (logo, imágenes)
